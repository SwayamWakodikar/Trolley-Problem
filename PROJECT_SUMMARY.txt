THE TROLLEY PROBLEM - ETHICS AND VALUES LAB
Complete Interactive Website
================================================================================

✅ PROJECT COMPLETED SUCCESSFULLY!

WHAT WAS CREATED
----------------

A fully functional, responsive website exploring the famous "Trolley Problem"
thought experiment in moral philosophy. All features work without any backend
- everything runs in the browser using localStorage.

PAGES & FEATURES
----------------

1. HOME PAGE (/)
   ✓ Engaging introduction to the trolley problem
   ✓ Beautiful animations using Framer Motion
   ✓ Clear call-to-action button
   ✓ Educational overview card

2. SIMULATION PAGE (/simulation)
   ✓ Animated SVG trolley moving on tracks
   ✓ Interactive decision-making (Pull Lever / Do Nothing)
   ✓ Real-time animation showing consequences
   ✓ Philosophical quotes after each decision
   ✓ Restart functionality
   ✓ All choices saved to localStorage

3. REFLECTION PAGE (/reflection)
   ✓ Statistics showing all past decisions
   ✓ Visual breakdown (times pulled vs. times did nothing)
   ✓ Decision history with timestamps
   ✓ Three guided reflection questions with text areas
   ✓ Save individual reflections to localStorage
   ✓ View past saved reflections
   ✓ "Reset All Data" button to clear everything

4. ABOUT PAGE (/about)
   ✓ Detailed explanation of three ethical theories:
     - Utilitarianism (consequences-based)
     - Deontology (duty-based)
     - Virtue Ethics (character-based)
   ✓ Real-world applications section:
     - Self-driving cars
     - Military decisions
     - Medical triage
     - AI ethics
   ✓ Variations of the trolley problem
   ✓ Further reading resources with links

5. QUIZ PAGE (/quiz)
   ✓ 4-question personality quiz
   ✓ Determines ethical thinking style
   ✓ Three possible results:
     - Utilitarian Thinker
     - Deontological Thinker
     - Virtue Ethics Thinker
   ✓ Visual score breakdown with progress bars
   ✓ Detailed result explanations
   ✓ Retake functionality

DATA PERSISTENCE
----------------

All user data stored in browser's localStorage:
- Decision history (pull lever vs. do nothing)
- Timestamps for each decision
- Reflection responses
- Quiz results

No backend required - works completely offline after initial load!

HOW TO RUN
----------

1. Install dependencies:
   npm install

2. Start development server:
   npm run dev

3. Open browser to: http://localhost:5173

4. Build for production:
   npm run build
